World of Warcraft on Linux

This guide assumes Battle.net is already installed using Steam Proton.

Tested versions

  • Retail
  • Proton Experimental & Proton 10

Recommended Proton

  • Proton 10

Installation notes

  • Install WoW normally from Battle.net
  • Do not move the install directory
  • Let Steam manage Proton
  • Guide to Install Battle.net

Addons

  • Addons work exactly like Windows.
  • Location: World of Warcraft/_retail_/Interface/AddOns
  • Or replace \_retail_ with _classic_ or _classic_era_
  • No extra tools required.
  • Guide to install Addons

Performance tips

  • Use DX12 (default)
  • Enable Steam shader pre-caching
  • Use borderless fullscreen
